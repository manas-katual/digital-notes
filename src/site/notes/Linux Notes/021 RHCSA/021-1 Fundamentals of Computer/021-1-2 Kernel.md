# Kernel

- Kernel is the engine of the operating system
- Features of an OS like
	>&rarr; Stability
	&rarr; Reliability
	&rarr; Flexibility
	&rarr; Security
	&rarr; Performance
	&rarr; Hardware Support
	&rarr; Software Support
	&rarr; Architecture
- This all depends on kernel	
- If kernel crashes, the entire OS will crash.
